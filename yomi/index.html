How it works

No API. No bot account. No webhook.

沒有 API、沒有 bot 帳號、沒有 webhook。

yomi logs in as a secondary device on your own account, speaks LINE's protocol directly, and hands the result to your agent — all on your machine.

yomi 以「次要裝置」登入你自己的帳號、直接講 LINE 的協定，再把結果交給你的 agent——全部在你的機器上完成。

01
Pair as a device
配對為裝置

Log in as a secondary device on your own LINE account. No bot, no official API.

以次要裝置登入你自己的 LINE 帳號。不用 bot、不用官方 API。

02
Speak the protocol
直接講協定

yomi talks LINE's TCompact-over-HTTPS protocol directly — no browser in the loop.

yomi 直接講 LINE 的 TCompact-over-HTTPS 協定——過程中沒有瀏覽器。

03
Decrypt locally
本機解密

Letter Sealing (E2EE) messages and media are decrypted on your machine — never in the cloud.

Letter Sealing（E2EE）的訊息與媒體都在你的機器上解密——不經過雲端。

04
Expose to any agent
開放給任何 agent

A small stdio MCP server. Point Claude — or any MCP client — at it, and it just works.

一個小巧的 stdio MCP server。把 Claude、或任何 MCP client 指過來，就能用。

用你自己的方式，補上 LINE 的進度。

What it does

Read & reply

讀取與回覆

Read messages, reply, send images, and mention people — the full loop, from your agent.

讀訊息、回覆、傳圖、標註成員——從你的 agent 完成整個循環。

Search everything

搜尋全部

Search across every conversation in your whole history — locally, in an instant.

在你的全部歷史裡跨對話搜尋——本機、瞬間完成。

End-to-end encrypted

端對端加密

Letter Sealing messages and media are decrypted on your machine. Nothing leaves it.

Letter Sealing 的訊息與媒體在你的機器上解密。沒有東西離開它。

Local-first

本機優先

A stdio MCP server that runs beside your agent. No server to host, no data to hand over.

一個跟 agent 並肩執行的 stdio MCP server。不用架站，也不交出資料。

No clone, no build

免 clone、免 build

Runs straight from npm via bunx. One line and your agent can read LINE.

透過 bunx 直接從 npm 執行。一行，你的 agent 就能讀 LINE。

Works with any MCP client

相容任何 MCP client

Claude Desktop, Claude Code, or anything that speaks the Model Context Protocol.

Claude Desktop、Claude Code，或任何會講 Model Context Protocol 的工具。

Honest note

Unofficial. yomi is an independent personal project, not affiliated with or endorsed by LINE. Using it may be against LINE's Terms of Service, and running an additional client on your account carries a risk of rate-limiting or suspension. It's intended for reading your own account. Use it at your own risk.

非官方。yomi 是獨立的個人專案，與 LINE 無任何關聯、也未獲其背書。使用它可能違反 LINE 的服務條款，且在帳號上多跑一個 client 有被限流或停權的風險。它是用來讀取你自己的帳號。請自行承擔風險。
