Six tools, one philosophy

六個工具，一套理念

clone it · read it · run it

Every name is a native Japanese word that carries layers of meaning through kanji, and what it describes is always human perception and care. As AI agents and MCP become everyday, these tools choose the human side — all MIT open source, local-first, privacy-first.

每個名字都是一個日文和語、以漢字承載多層意象，描述的永遠是人的知覺與關懷。當 AI agent 與 MCP 成為日常，這些工具選擇站在人這邊——全部 MIT 開源、本機優先、隱私優先。

yomi
読み · 讀／詠／黄泉

Read and write your LINE from any AI agent — read the messages, and read the context too. An MCP server with E2EE, decrypted locally.

從任何 AI agent 讀寫你的 LINE——讀訊息，也讀懂情境。E2EE 本機解密的 MCP server。

claude mcp add yomi -- bunx @rikaidev/yomi
toki
時 · 時間／時刻

Automatic time tracking for developers — no timer to press. It reads the context from your IDE and git, so time understands itself.

開發者自動時間追蹤，不需按下計時。從 IDE 與 git 讀出脈絡，時間自己被理解。

brew install RikaiDev/tap/toki
mesen
目線 · 視線／視角

UX testing through simulated user perspectives — see your interface through someone else's eyes. Persona-driven and grounded in evidence.

以模擬使用者視角做 UX 測試——用別人的眼睛看你的介面。persona 驅動、有據可依。

npm install -D mesen
kaido
海道 · 通海的路

Your AI coach in the terminal — it diagnoses and teaches at once, so you learn why, not just how. Runs on a local LLM, offline-capable.

你的終端機 AI 教練——邊診斷邊教學，學為什麼，不只是怎麼做。本機 LLM 可離線。

cargo install --git https://github.com/RikaiDev/kaido
mimamori
見守り · 守望／守護

A Discord bot that watches over your team's mood — gentle attention, not surveillance. It understands context across channels.

守護團隊溝通氛圍的 Discord bot——溫柔地看顧，而非監控。跨頻道理解語境。

git clone https://github.com/RikaiDev/mimamori
minori
実り · 結果／收成

An AI collaboration platform for farming — log the fields by voice, forecast the harvest, and connect farmers with buyers over LINE. So the work bears fruit.

AI 農業協作平台——語音記錄田間、預測收成，透過 LINE 連結農戶與買家。讓耕耘有結果。

bun install · bun run dev

Why open source

The code is the proof.
Not slides, not a demo, not a Twitter thread.

程式碼，即證明。
不是投影片，不是 demo，不是 Twitter thread。

Everyone ships with AI now; the only difference is whether yours can be cloned, read, and verified.
If AI-native development works, you should be able to see it in the commits.

如今人人都用 AI 出貨；差別只在於，你的能不能被 clone、被讀、被驗證。
If AI-native development works, you should be able to see it in the commits.

clone

All MIT. Clone it and it's yours.

全部 MIT。抓下來就是你的。

read

Every decision lives in the commits, laid open for you to see.

每個決定都在 commit 裡，攤開給你看。

run

Local-first, privacy-first — your data stays on your machine.

本機優先、隱私優先，資料留在你機器上。

break

Open an issue, send a PR. An honest failure beats a faked success.

開 issue、送 PR。誠實的錯誤，勝過偽造的成功。
